Yii Countdown Timer Widget
===

Version 0.3.0

## Requirements

Yii 1.1 or above, PHP >= 5

## Install

To install this widget
-Download this package or get it via composer.

        'components'=>array(
                'countdown' => array(
                        'class'=>'vendor.quexer69.yii-countdown.CountDownWidget'
                ),
        ),
